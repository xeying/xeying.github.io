文件系统管理

常用的硬盘管理命令

df命令

df -ahT 查看硬盘空间大小（需要重启释放，统计的剩余空间大小是准确的）

du命令

du -h 查看目录大小（统计的文件大小是准确的）

fsck文件系统修复命令

fsck -y /dev/sdb1 自动修复

显示磁盘状态dumpe2fs

centos6时用 dumpe2fs 命令，在 centos7时 用 xfs_growfs来查看文件系统信息

判断文件类型

type 命令名 判断命令类型

file 文件名 判断文件类型

fdisk命令手工分区

fdisk -l 查看系统所有硬盘及分区
